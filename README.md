# kalkulator_bat

Prosty kalkulator w formie skryptu batch dla systemu Windows. Umożliwia wykonywanie podstawowych działań matematycznych na dwóch liczbach.

## Wymagania

- System Windows
- Interpreter poleceń CMD (standardowo dostępny w Windows)

## Uruchomienie

W katalogu z plikiem uruchom:

```batch
kalkulator.bat
```

Lub dwuklik na plik `kalkulator.bat` w Eksploratorze plików.

## Użycie

1. Podaj pierwszą liczbę.
2. Podaj drugą liczbę.
3. Wybierz działanie:
   - **1** — dodawanie
   - **2** — odejmowanie
   - **3** — mnożenie
   - **4** — dzielenie
4. Program wyświetli wynik i wróci do menu (naciśnij dowolny klawisz, aby kontynuować).

## Uwagi

- Dzielenie jest całkowitoliczbowe (wynik zaokrąglany w dół).
- Przy dzieleniu przez 0 może wystąpić błąd.
- Po wybraniu nieprawidłowej opcji (innej niż 1–4) wyświetlany jest komunikat „Nieprawidlowa opcja!” i program wraca do menu.
